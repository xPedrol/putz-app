Bem-vindo a Putz Filmes

Obrigado por se juntar a Putz Filmes. Este é um guia rápido para você começar a usar o sistema.

Você está no Painel de controle. Nesta tela temos informações sobre seu portfólio, inscrições as quais você se cadastrou em uma lista de todos os projetos com sua participação.

A plataforma possui dois cabeçalhos. Como pode ser visto, o primeiro é o mais clássico, com um ícone para alteração de tema e sua foto de perfil. Logo abaixo deste, se encontra o sub-cabeçalho, que possui links para as várias páginas do sistema.

Como utilizar a plataforma?

Aqui você verá um passo a passo de como se preparar para concorrer às oportunidades de trabalho.

Como freelancer, você pode acessar a página de projetos, onde é possível ver todos os projetos aos quais você foi aprovado. Mas primeiro, é preciso organizar bem seu portfólio, para isso, basta clicar no link de portfólio, depois no botão de adição perto da tabela e começar a enviar seus dados. Logo depois, você terá que se cadastrar às oportunidades, o que pode ser feito clicando no link de Inscrições e depois pesquisando por uma que atenda aos seus requisitos. E lembre-se que você será escolhido conforme a qualidade de seu portfólio.

Após aprovação em determinada oportunidade de trabalho, você receberá autorização para entrar em um projeto. Entretanto, só será possível acessar a etapa a qual vc se cadastrou, uma vez que outros freelancers trabalham em outras etapas do mesmo projeto.

Isso é tudo!

Você está pronto para utilizar a plataforma :).

Caso ainda tenha alguma dúvida, existem botões de ajuda que você pode clicar para obter mais informações. O sub-cabeçalho mesmo possui um. Clicando nele um guia explicando cada página será exibido. Boa sorte!
